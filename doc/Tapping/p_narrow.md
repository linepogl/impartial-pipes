## p_narrow

Returns a partial function that checks if the input is an instance of the specified class.

### Syntax
```php
p_narrow(
  class-string
)
```

### Examples

```php
new DateTime()
|> p_narrow(DateTime::class)
//= new DateTime()
```
```php
new DateTime()
|> p_narrow(DateInterval::class)
// throws TypeError
```
