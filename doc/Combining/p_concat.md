## p_concat

Returns a partial function that merges two iterables.

### Syntax
```php
p_concat(
  iterable<TOtherKey, TOtherValue>,
)
```

### Examples
Concatenating two iterables
```php
[1, 2]
|> p_concat([3, 4])
//= [1, 2, 3, 4]
```
```php
['a' => 1, 'b' => 2]
|> p_concat(['b' => 22,'c' => 3])
//= [1, 2, 22, 3]
```
