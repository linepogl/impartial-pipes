## p_assoc_skip

Returns a partial function that skips the first n elements of an iterable, preserving the keys.

### Syntax
```php
p_assoc_skip(
  int,
)
```

### Examples
Skip elements
```php
[1, 2, 3, 4]
|> p_assoc_skip(2)
//= [2 => 3, 3 => 4]
```
```php
['a' => 1, 'b' => 2, 'c' => 3, 'd' => 4]
|> p_assoc_skip(2)
//= ['c' => 3, 'd' => 4]
```
