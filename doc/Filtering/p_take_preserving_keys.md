## p_take_preserving_keys

Returns a partial function that takes the first n elements of an iterable, preserving the keys.

### Syntax
```php
p_take_preserving_keys(
  int,
)
```

### Examples
Take elements
```php
[1, 2, 3, 4]
|> p_take_preserving_keys(2)
//= [1, 2]
```
```php
```
['a' => 1, 'b' => 2, 'c' => 3, 'd' => 4]
|> p_take_preserving_keys(2)
//= ['a' => 1, 'b' => 2]
```php
