## p_compact_preserving_keys

Returns a partial function that filters null values out of an iterable, preserving the keys.

### Syntax
```php
p_compact_preserving_keys()
```

### Examples
Filter out nulls
```php
['a' => 1, 'b' => null, 'c' => 3]
|> p_compact_preserving_keys()
//= ['a' => 1, 'c' => 3]
```
