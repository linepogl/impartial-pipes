## p_assoc_while_not_null

Returns a partial function that takes elements of an iterable, while they are not null, preserving the keys.

### Syntax
```php
p_assoc_while_not_null()
```

### Examples
Take elements until a null is encountered
```php
['a' => 1, 'b' => 2, 'c' => null, 'd' => 4]
|> p_assoc_while_not_null()
//= ['a' => 1, 'b' => 2]
```
