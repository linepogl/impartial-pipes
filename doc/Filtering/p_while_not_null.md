## p_while_not_null

Returns a partial function that takes elements of an iterable, while they are not null.

### Syntax
```php
p_while(
  [preserveKeys: bool = false,]
)
```

### Examples
Take elements until a null is encountered
```php
[1, 2, null, 4]
|> p_while_not_null()
//= [1, 2]
```
```php
['a' => 1, 'b' => 2, 'c' => null, 'd' => 4]
|> p_while_not_null(preserveKeys: true)
//= ['a' => 1, 'b' => 2]
```
