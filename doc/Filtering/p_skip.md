## p_skip

Partial function to skip the first n elements of an iterable.

### Syntax
```
p_skip(
  int
  [, preserveKeys: bool = false]
)
```

### Examples
Skip elements
```
[1, 2, 3, 4]
|> p_skip(2)
//= [3, 4]
```
```
[1, 2, 3, 4]
|> p_skip(2, preserveKeys: true)
//= [2 => 3, 3 => 4]
```
```
['a' => 1, 'b' => 2, 'c' => 3, 'd' => 4]
|> p_skip(2)
//= [3, 4]
```
```
['a' => 1, 'b' => 2, 'c' => 3, 'd' => 4]
|> p_skip(2, preserveKeys: true)
//= ['c' => 3, 'd' => 4]
```
