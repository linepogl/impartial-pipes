## p_skip

Returns a partial function that skips the first n elements of an iterable.

### Syntax
```php
p_skip(
  int,
)
```

### Examples
Skip elements
```php
[1, 2, 3, 4]
|> p_skip(2)
//= [3, 4]
```
```php
['a' => 1, 'b' => 2, 'c' => 3, 'd' => 4]
|> p_skip(2)
//= [3, 4]
```
