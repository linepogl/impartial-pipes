## p_filter_out_nulls

Partial function to filter out null values of an iterable.

### Examples

```
[1, 2, null, 3]
|> p_filter_out_nulls()
//= [1, 2, 3]
```
```
['a' => 1, 'b' => null, 'c' => 3]
|> p_filter_out_nulls(preserveKeys: true)
//= ['a' => 1, 'c' => 3]
```
