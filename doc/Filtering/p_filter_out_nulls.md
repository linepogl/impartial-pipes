## p_filter_out_nulls

Partial function to filter null values out of an iterable.

### Syntax
```php
p_filter_out_nulls(
   [preserveKeys: bool = false,]
)
```

### Examples
Filter out nulls
```php
[1, 2, null, 3]
|> p_filter_out_nulls()
//= [1, 2, 3]
```
```php
['a' => 1, 'b' => null, 'c' => 3]
|> p_filter_out_nulls(preserveKeys: true)
//= ['a' => 1, 'c' => 3]
```
