## p_assoc_compact

Returns a partial function that filters null values out of an iterable, preserving the keys.

### Syntax
```php
p_assoc_compact()
```

### Examples
Filter out nulls
```php
['a' => 1, 'b' => null, 'c' => 3]
|> p_assoc_compact(preserveKeys: true)
//= ['a' => 1, 'c' => 3]
```
